### 颜色单位
- 颜色名[英文]
- RGB值
 - 通过三种颜色的不同浓度来调配出不同的颜色
 - `R = red` `G = green` `B = blue`
 - 每一种颜色范围在0 - 255(0% - 100%)之间
 - 语法:RGB(R,G,B)
- RGBA值
 - A = 透明值
 - `1 = 完全不透明` `0 = 完全透明`
 - 语法:RGBA(R,G,B,A)
- 十六进制的RGB值
 - 颜色浓度 00 - ff
 - 语法:#RGB
 - 两位两位重复，可以简写
  - #aa0102 = a0102
- HSL值、HSLA值
 - `H = 色相` `S = 饱和度` `L = 亮度` `A = 透明度`
 - H:0-360
 - S:颜色浓度 0 - 100
 - L:亮度 0 -100
