# Markdown语法
### 标题
两种形式
> 1. 使用=和-标记一级和二级标题
	- 一级标题
	```========```
	- 二级标题
	```--------```
> 2. 使用#，可以表示1-6极标题
	- # 一级标题
	- ## 二级标题
	- ### 三级标题
	- #### 四级标题
	- ##### 五级标题
	- ###### 六级标题
### 段落
> 段落前后要有空行
### 区块引用
```
在段落的每行或者第一行使用>符号，可以多个嵌套。
> 区块应用
>> 嵌套引用
```
> 区块引用
>> 嵌套引用
### 代码区块
> 每行4个空格或者一个制表符
> 或者\`\`\`内容\`\`\`
> 注意需要和普通段落直接存在空行
```
	void main()
	{
		printf("hello,Markdown.");
	}
```
### 强调
> 在强调内容两侧分别加上\*或者\_
> *斜体*
> _粗体_
### 列表
> 使用\·、\+、\-标记无序列表
_注意:_标记后面最好有一个空格或者制表符，若不在引用区块中，必须和前方段落之间存在空行。
有序标记这是在.前面加数字
> - one
> - two
> - 3

1. 1
2. 2
3. 3
### 分割线
三个以上*、-、_
### 链接
两种：_行内式_和_参考式_
```
[链接内容](http://wezhon.com)
```
> [my blog](http://wezhon.com)
```
[china][1]
[world][2]


#结尾
[1]:http://china.cn.com
[2]:http://world.com
```
